### 北京邮电大学“网络搜索原理”课程(2020)
- 课程名称：网络搜索原理
- 教师：陈光 ([@爱可可-爱生活](https://weibo.com/fly51fly))
- 时间：周三上午 8:00-9:35
- 地点：北京邮电大学教三楼132

本课程旨在通过实战的方式，理解搜索引擎背后的基本技术原理，并将其应用于解决实际问题。本课程延续《Python编程与实践》的讲课风格，主打编程驱动、问题驱动的学习体验，用进化视角看技术更迭，培养搜索思维，开阔解决问题的思路。
